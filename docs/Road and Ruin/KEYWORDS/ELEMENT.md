MAIN COMPONENTS OF AN ELEMENT

- DAMAGE TYPE
- EFFECT OVER TIME
- DEBILITATION
- ENFORCEMENT
- ARRANGE PATTERN
- IMBUE
- COAT
- CHANGE OF STATE
- ENVIRONMENTAL IMPACT

- SIZE
- SHAPE
- DURATION
- INTENSITY
- RANGE
- PRECISION
- RESISTANCE
- REDUCTION
- INFUSION
- ANIMATION
- GLEANING
- SENSING
- COMMUNION
